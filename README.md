# postal-codes.js
Provide postal code validation for javascript


## Still in development / no release yet


### Validation rules
1. Characters " " (space) and "-" (dash) are removed from the passed string
2. Case doesn't matter

### Testing with mocha
    $ npm test
    $ npm run coverage
